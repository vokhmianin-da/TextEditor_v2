Данный текстовый редактор позволяет вводить текст, сохранять текст в файл формата (.txt), а также открывать файлы формата (.txt) и редактировать их.

Есть возможность переназначения горячих клавиш. Для этого необходимо выбрать нужный пункт меню "Назначить клавиши" и нажать новую комбинацию.

Текстовый редактор позволяет Менять шрифты и выравнивание текста.

При помощи меню "Оформление" можно переключаться между русским и английским языком, а также выбирать светлую или темную тему приложения.

Пункт меню "Режим" позволяет открывать файлы только для чтения. Для этого необходимо проставить галочку в меню "Только для чтения"

Для того, чтобы выбрать файл, который требуется открыть, нажмите кнопку "Файл"->"Открыть".

Для того, чтобы сохранить файл, нажмите кнопку "Файл"->"Сохранить" или "Файл"->"Сохранить как".

Для создания нового файла необходимо выбрать меню "Файл"->"Создать".

Текстовый редактор поддерживает возможность одновременного открытия нескольких документов.

Для вывода выбранного документа на печать необходимо выбрать меню "Файл"->"Печать".
